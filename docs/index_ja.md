# Bridgy ヘルプ（日本語）

最終更新日：2026-02-23

## このアプリについて
Bridgy は、難聴・発話が困難な方、言語の違いなどによりコミュニケーションが難しい場面で、対面での会話を補助（橋渡し）する iPad 向けアプリです。

- 手書き（指 / Apple Pencil）
- キーボード入力
- 音声 → 文字（Speech）
- 翻訳表示（Translation / Translate）

**オフラインで利用できます（ただし音声認識/翻訳は Apple の仕組みによりサーバー処理になる場合があります）。**

---

## 使い方（基本）
1. アプリを起動します  
2. 入力方法（手書き / 文字入力 / 音声）を選びます  
3. 必要に応じて翻訳を表示します  
4. 画面を相手に見せて、対面の会話を補助します  

---

## 注意（重要）
- **高リスク用途（医療・救急/安全・法律・金融）には使用しないでください。**
- **個人情報や機微情報（住所・口座番号・パスワード等）は表示しないでください。**
- **スクリーンショット／画面録画／共有は慎重に扱ってください。**
